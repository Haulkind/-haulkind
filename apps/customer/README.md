# Haulkind Customer App

Expo mobile app for Haulkind customers.

## Features
- Auth (login/signup)
- Job creation flow (Haul Away + Labor Only)
- Real-time tracking map
- Extension approval
- Receipt & support

## Setup
```bash
cd /home/ubuntu/haulkind/apps/customer
pnpm install
pnpm web
```
